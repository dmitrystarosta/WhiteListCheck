## Что уже реализовано

- Ручная проверка трёх групп сайтов, вердикты 4 типов + UNKNOWN
- Настройка списков (v0.3): шестерёнка → экран «Списки сайтов»;
  добавление по домену (латиницей), удаление (минимум 1 сайт в группе),
  диалог подтверждения с объяснением последствий (у каждой группы свой
  текст), сброс к стандартным. Хранение: SharedPreferences, ключ
  custom_lists, JSON того же формата, что remote config; пользовательский
  список имеет приоритет над REMOTE_CONFIG_URL; фоновая проверка
  использует те же списки. Кириллические домены не принимаются
  (осознанное упрощение)
- **Android TV, этап 1 (v0.3.1):** векторный баннер banner.xml 320×180
  (иконка + название приложения, текст переведён в кривые), в манифесте
  LEANBACK_LAUNCHER, android:banner, uses-feature touchscreen/leanback
  required=false. Приложение видно в лаунчере ТВ, запускается
  и проверяет с пульта
- **Android TV, этап 2 (v0.3.2):** подсветка фокуса для D-pad — функция
  Modifier.tvFocusHighlight (рамка 2px #3F51B5 + полупрозрачный фон)
  на всех интерактивных элементах обоих экранов; onFocusChanged должен
  стоять ДО clickable в цепочке модификаторов. На телефоне не
  проявляется
- **Android TV, этап 3 (v0.3.3):** TRANSPORT_ETHERNET в networkType,
  сообщение «Ethernet (кабель)» с пояснением в UI. ТВ-поддержка
  базово полная
- Фоновая проверка (WorkManager, 15 мин) с уведомлениями о смене режима,
  переключатель в UI, запрос POST_NOTIFICATIONS на Android 13+
- Тип сети с адаптивными сообщениями («нет сети», Wi-Fi с предупреждением
  про 3G/4G-роутер, Ethernet)
- Кликабельные названия сайтов, русские описания ошибок, сноска про
  механизмы блокировок + сноска про Instagram
- Подвал: версия (ссылка на релизы) + © 2026, Dmitry Starosta
- Иконка «чебурнет» (вектор в приложении, PNG 512×512 для магазинов)
- CI (GitHub Actions): release-APK, подписанный ключом из секретов;
  APK автоматически именуется WhiteListCheck_v<версия>.apk — версия
  извлекается из build.gradle.kts шагом Rename в build.yml.
  Провалившийся подход: archivesBaseName в build.gradle.kts — свойство
  удалено в новых Gradle (раннер использует Gradle 9.x), не повторять
- README с SEO-абзацем, топики репозитория, LICENSE (MIT)
- PRIVACY.md в корне репозитория (данные не собираются) — для магазинов

### Решение от 08.07.2026: состав группы Б

Drive2 и Banki.ru исключены: по сообщениям пользователей (Чувашия, Билайн)
открывались при включённом белом списке. Проблема системная: перечень
Минцифры (>500 ресурсов на май 2026) еженедельно пополняется популярными
российскими ресурсами, различается у операторов, банки добавляются
поэтапно. Контрольные сайты группы Б — популярные зарубежные
незаблокированные ресурсы. Настройка списков в v0.3 — вторая часть
решения: пользователь адаптирует списки под свой регион.
